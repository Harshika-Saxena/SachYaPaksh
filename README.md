# SachYaPaksh
SachYaPaksh is an AI-powered media analysis tool designed to detect and flag bias in real-time. By combining Natural Language Processing (NLP) and fined-tuned DistilBERT model, the platform helps users distinguish between objective reporting and biased influence.

* **Bias Detection:** Calculates a "Bias Confidence Score" based on narrative portrayal and sentiment analysis.

* **TrueView:** A visual analysis tool using **OCR** to extract text from posters and ads to find hidden representation gaps.
